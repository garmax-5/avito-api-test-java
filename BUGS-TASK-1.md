## [TestBUG-1] Не подгружается изображение кнопки очистки содержимого формы поиска

**Описание:**  
На странице «Вакансии» в Avito у кнопки очистки содержимого формы поиска вместо изображения отображается серый фон.

**Шаги воспроизведения:**
1. Зайти на страницу «Вакансии» в Avito.
2. В форме поиска ввести текст.

**Фактический результат:**  
Страница «Вакансии» в Avito с кнопкой очистки содержимого формы поиска на сером фоне (без изображения).

**Ожидаемый результат:**  
Страница «Вакансии» в Avito с кнопкой очистки содержимого формы поиска с изображением.

**Окружение:** Неизвестно

**Приоритет:** Low / Medium

**Комментарий:**  
- Если баг не влияет на основной функционал страницы и не блокирует пользовательские сценарии, и затрагивает только загрузку изображения (а не работу кнопки), то баг носит визуальный характер - приоритет **Low**.  
- Если баг затрагивает не только отображение изображения, но и функциональность кнопки очистки, то приоритет **Medium**, поскольку ошибка частично нарушает корректную работу страницы, однако не блокирует основной пользовательский сценарий.

## [TestBUG-3] При выборе «Санкт-Петербург» в фильтре «Город» в результатах отображаются вакансии из других городов

**Описание:**  
На странице «Вакансии» в Avito при выборе в фильтре «Город» значения «Санкт-Петербург» в итоговом списке вакансий отображаются вакансии из других городов.

**Шаги воспроизведения:**
1. Зайти на страницу «Вакансии» в Avito.
2. В форме поиска ввести «Аналитик».
3. В фильтре «Город» выбрать «Санкт-Петербург».
4. В фильтре «Направление» выбрать «Data Science».

**Фактический результат:**  
В итоговом списке вакансий отображаются позиции из всех городов, включая те, которые не соответствуют выбранному городу («Санкт-Петербург»).

**Ожидаемый результат:**  
В итоговом списке вакансий отображаются только позиции, которые включают выбранный город («Санкт-Петербург»).

**Окружение:** Неизвестно

**Приоритет:** High

**Комментарий:**  
Баг нарушает основной сценарий поиска вакансий по выбранному городу: при выборе Санкт-Петербурга отображаются вакансии из других городов, и пользователь не может получить корректные результаты. Поскольку поиск по городу - ключевая функция сайта, баг требует срочного исправления.
